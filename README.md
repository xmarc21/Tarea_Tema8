# Tarea Tema 8 - DateTime (SMR)

Aplicación de consola en C# que pide nombre completo y fecha de nacimiento, y genera un informe personalizado con:
- Nombre extraído
- Edad exacta
- Signo del zodiaco
- Días hasta el próximo cumpleaños
- Fecha de nacimiento en formato largo (es-ES)

**Requisitos cumplidos:**
- Validación con `TryParse` + bucle while
- CultureInfo("es-ES")
- Rechazo de fechas futuras
- Uso de `IndexOf` y `Substring` para extraer el primer nombre

Proyecto realizado para la asignatura **Entornos de Desarrollo**.
